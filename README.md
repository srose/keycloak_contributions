# Contributing to Keycloak

This is dedicated to learning how to do contributions to [Keycloak](https://github.com/keycloak/keycloak).

If these things eventually find their way inside [Keycloak](https://github.com/keycloak/keycloak) itself, they will be removed from here.

The sessions will be part of the meetup [Keycloak Hour of Code](https://www.meetup.com/keycloak-hour-of-code)

Get in touch if you would like to do share something by creating issues or pull requests to this repository.
